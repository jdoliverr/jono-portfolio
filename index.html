Language Repetition - Simple fullstack app used to help in learning a new language. It uses the learning technique known as spaced repetition to teach you words you don't know. The more can correctly translate the word the less often you see it and so the less you can translate the word the more frequently you see it. This is a technique commonly used with flashcards and is implemented here in a similar way. On the technical side the linked list data structure is used for the words to learn, keeping track of correct answers and how many times in a row you translate correctly so it can move a word the appropriate amount back in the list.
